# creditoswebsite
creditoswebsite es un portal web diseñado para el proyecto Aprobación automática de solicitudes de crédito interno corporativo, utilizando datos provenientes del sistema Gestión de Cobros.El sitio se conecta a un flujo automatizado en Azure Logic Apps, que procesa la información y la envía a un modelo de IA alojado en Azure Foundry para generar decisiones automáticas basadas en reglas y análisis inteligente.Este portal forma parte de una arquitectura moderna construida sobre Azure Static Web Apps, integrada con GitHub, permitiendo despliegues continuos (CI/CD) y manteniendo el proyecto centralizado, escalable y fácil de mantener.
